Module 13: WLAN Configuration

13.1 – Remote Site WLAN Configuration

  • Wireless Routers: Combine AP, switch, router, NAT, DHCP, and QoS.
  • Always change default credentials (e.g., admin/admin).
  • Setup Steps:
    • Change admin password, configure DHCP, renew IP.
    • Set SSID, select non-overlapping channel, use WPA2 Personal with passphrase.
  • Extended Coverage:
    • Add APs with same SSID but different 2.4 GHz channels (1, 6, 11).
  • Advanced Features:
    • NAT: Translates private to public IP for internet.
    • QoS: Prioritizes voice and video traffic.
    • Port Forwarding: Opens specific ports for apps/services.

13.2 – Configure a Basic WLAN on the WLC

  • Topology: Lightweight APs managed via CAPWAP (no manual config).
  • WLC trunk interfaces carry VLANs to APs.
  • Steps:
    • Log in to WLC GUI (default IP/creds).
    • Create WLAN: Set SSID, enable, link to VLAN interface.
    • Use WPA2-PSK with AES encryption.
    • Monitor clients and traffic via WLC interface.

13.3 – Configure a WPA2 Enterprise WLAN on the WLC

  • Prerequisites:
    • RADIUS server for AAA (e.g., FreeRADIUS).
    • SNMP server for trap monitoring.
  • Steps:
    • Create vlan5 with DHCP scope (192.168.5.0/24).
    • Bind WLAN to vlan5, enable WPA2-Enterprise (AES + 802.1X).
    • Link RADIUS server (IP + shared secret).
    • Verify via client connections and RADIUS logs.

13.4 – Troubleshoot WLAN Issues

  • Troubleshooting Steps:
    • Identify issue (e.g., no connectivity).
    • Check client settings: IP config, NIC driver, security.
    • Inspect AP/WLC: Power, cabling, channel overlap.
  • Fixing Slow Networks:
    • Upgrade legacy 802.11b/g clients.
    • Split traffic: 5 GHz for streaming, 2.4 GHz for basic use.
    • Apply firmware updates to fix bugs and boost performance.

13.5 – Module Summary

  • Remote Sites: Wireless routers with NAT/QoS, mesh for extended coverage.
  • WLC Management: Centralized AP/VLAN/RADIUS control.
  • Security: WPA2-Enterprise for enterprises, WPA2-PSK for homes.
  • Troubleshooting Flow: Clients → AP/WLC → Optimize traffic & signal.
  • Key Points:
    • SNMP & RADIUS: Crucial for enterprise monitoring/authentication.
    • DHCP Scope: VLAN-specific addressing via WLC.
    • FlexConnect: APs switch locally when WLC is unreachable.
    • Firmware: Regular updates patch vulnerabilities and bugs.
