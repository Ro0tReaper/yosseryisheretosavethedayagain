Module 12: WLAN Concepts

12.1 – Introduction to Wireless

  • WLANs provide wireless access for mobility and adaptability in homes, offices, and campuses.
  • Wireless Types:
    • WPAN: Short-range (Bluetooth, Zigbee)
    • WLAN: Medium-range (Wi-Fi)
    • WMAN: City-wide (WiMAX)
    • WWAN: Global (Cellular, Satellite)
  • Technologies: Bluetooth (BR/EDR, BLE), WiMAX, GSM/CDMA, Satellite (needs line of sight)
  • 802.11 Standards:
    • Vary by frequency (2.4 vs 5 GHz), speed (e.g., 802.11ac up to 1.3 Gb/s), compatibility
    • 802.11ax (Wi-Fi 6): Adds support for 1 GHz and 7 GHz bands
  • Frequencies:
    • 2.4 GHz: Longer range, more interference (802.11b/g/n/ax)
    • 5 GHz: Faster, less congestion (802.11a/n/ac/ax)
  • Standards Bodies:
    • ITU: Regulates spectrum
    • IEEE: Defines 802.11
    • Wi-Fi Alliance: Ensures interoperability

12.2 – WLAN Components

  • Wireless NIC: Integrated or USB adapters for wireless access.
  • Home Routers: Combine AP, switch, and router functions.
  • Access Points:
    • Autonomous: Manually configured standalone APs
    • Controller-based: Lightweight APs managed by WLC using CAPWAP
  • Antennas:
    • Omnidirectional: 360° signal (e.g., home/office)
    • Directional: Focused signal (e.g., Yagi, parabolic)
    • MIMO: Multiple antennas (up to 8) for bandwidth

12.3 – WLAN Operation

  • Modes:
    • Infrastructure (AP-based)
    • Ad hoc (peer-to-peer)
    • Tethering (mobile hotspot)
  • Topologies:
    • BSS (Basic Service Set): Single AP
    • ESS (Extended Service Set): Multiple APs on wired LAN
  • CSMA/CA: Uses RTS/CTS for collision avoidance
  • Client Association: Discover → Authenticate → Associate
  • Scanning:
    • Passive: AP sends beacons
    • Active: Client sends probes

12.4 – CAPWAP Operation

  • CAPWAP: Protocol for WLC to manage APs with IPv4/IPv6 and DTLS encryption.
  • Split MAC:
    • AP handles: Beacons, acknowledgments, encryption
    • WLC handles: Authentication, roaming, translation
  • DTLS: Encrypts control plane; data encryption requires license
  • FlexConnect:
    • Connected Mode: WLC manages
    • Standalone Mode: AP switches locally if WLC unavailable

12.5 – Channel Management

  • Interference Mitigation:
    • DSSS: 802.11b
    • FHSS: Legacy 802.11
    • OFDM: 802.11a/g/n/ac
  • Channel Planning:
    • 2.4 GHz: Non-overlapping channels 1, 6, 11
    • 5 GHz: More 20 MHz-wide channels (e.g., 36, 48, 60)
  • Deployment Considerations: Coverage, user density, transmit power

12.6 – WLAN Threats

  • Main Threats:
    • Data interception (unencrypted traffic)
    • Rogue APs (unauthorized MITM)
    • DoS via interference or malicious intent
    • Evil Twin (fake AP with same SSID)
  • Countermeasures:
    • Monitor for rogue APs
    • Use encryption and authentication

12.7 – Secure WLANs

  • Early Protections:
    • SSID Cloaking (not effective alone)
    • MAC Filtering (easily bypassed)
  • Authentication:
    • Open: No password (public)
    • Shared Key:
      • WEP (insecure)
      • WPA (TKIP)
      • WPA2 (AES)
      • WPA3 (SAE -- stops brute forces/OWE -- encrypt all wireless traffic.)
  • Encryption:
    • TKIP: Legacy (WPA)
    • AES/CCMP: WPA2
    • SAE/DPP: WPA3
  • Enterprise Security:
    • 802.1X/EAP with RADIUS
    • Requires IP, UDP ports 1812(Authentication)/1813(Accounting) also can be 1645/1646, shared key
